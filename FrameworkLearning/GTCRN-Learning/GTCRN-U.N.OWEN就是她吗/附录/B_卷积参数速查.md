# 附录 B：卷积参数速查

## B.1 参数缩写对照表

| 缩写 | 全称 | 含义 | 常见值 |
|------|------|------|--------|
| k | kernel_size | 卷积核大小 | 1, 3, 5, 7 |
| s | stride | 步长 | 1, 2 |
| p | padding | 填充 | 0, 1, 2 |
| d | dilation | 空洞率 | 1, 2, 4, 8 |
| g | groups | 分组数 | 1, C（depthwise） |

---

## B.2 各参数的作用

### kernel_size (k)：卷积核覆盖多大范围

```
k=1: 只看当前位置（逐点卷积，只混合通道）
     ●

k=3: 看 3×3 区域
     █ █ █
     █ ● █
     █ █ █

k=(1, 5): 时间轴 1，频率轴 5
          █ █ ● █ █   只看当前时间帧的 5 个频点
```

**为什么 GTCRN 用 (1,5) 做下采样**：下采样卷积只需要在频率轴上聚合信息，时间轴保持不变，所以时间轴用 k=1。

### stride (s)：每次移动多少，决定输出尺寸

```
输入: 8 个位置
      1 2 3 4 5 6 7 8

s=1: 每次移动 1，输出 8 个（尺寸不变）
     ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓

s=2: 每次移动 2，输出 4 个（下采样 2 倍）
     ↓   ↓   ↓   ↓
```

输出尺寸公式：`out = (in + 2×p - k) / s + 1`

### padding (p)：在边缘补零，控制输出尺寸

```
p=0: 不填充，输出比输入小
p=k//2: 保持输出尺寸不变（当 s=1 时）
```

**因果 padding**：非因果卷积左右对称填充，因果卷积只在左边填充 `(k-1)×d`，右边不填充。这样每个位置只能看到过去的信息。

### dilation (d)：卷积核内部的间隔，扩大感受野

```
k=3, d=1（普通卷积）:
     █ █ █     感受野 = 3

k=3, d=2（空洞卷积）:
     █   █   █   感受野 = 5

k=3, d=4:
     █       █       █   感受野 = 9
```

感受野公式：`receptive = 1 + (k-1) × d`

**为什么用空洞卷积**：不增加参数，不增加计算量，但感受野指数增长。GTCRN 用膨胀率 [1,2,4,8,4,2] 把感受野扩大到 43 帧（约 430ms）。

### groups (g)：把通道分组独立卷积

```
普通卷积 (g=1):
     所有输入通道 → 所有输出通道
     参数: C_in × C_out × k²

分组卷积 (g=2):
     前半输入 → 前半输出
     后半输入 → 后半输出
     参数: C_in × C_out × k² / g

Depthwise 卷积 (g=C_in):
     每个通道独立卷积
     参数: C_in × k²
```

## B.3 深度可分离卷积

标准卷积 = 空间特征提取 + 通道混合，一步完成。
深度可分离卷积把这两件事拆开：

**第一步：深度卷积（Depthwise）**
- groups = C_in，每个通道独立做空间卷积
- 参数：C × k²

**第二步：逐点卷积（Pointwise，即 1×1 卷积）**
- 只混合通道，不看空间位置
- 参数：C_in × C_out

**节省比例**：

| 卷积类型 | 参数量 | 节省 |
|---------|--------|------|
| 标准卷积 | C_in × C_out × k² | — |
| 深度可分离 | C_in × k² + C_in × C_out | 约 1/k²（当 C_out >> k² 时约 1/C_out） |
| 实际（C=32, k=3） | 32×9 + 32×32 = 1312 vs 32×32×9 = 9216 | **约 7 倍** |

## B.4 尺寸计算公式

### 标准卷积（含空洞）

```
out = floor((in + 2×p - d×(k-1) - 1) / s + 1)
```

### 常用简化

| 条件 | 结果 |
|------|------|
| s=1, p=k//2, d=1 | out = in（尺寸不变） |
| s=2, p=k//2, d=1 | out ≈ in/2（下采样） |
| s=1, p=d×(k//2), d=d | out = in（空洞卷积，尺寸不变） |

### 转置卷积（上采样）

```
out = (in - 1) × s - 2×p + k + output_padding
```

当 s=2, p=k//2 时：`out ≈ in × 2`

## B.5 GTCRN 中的实际例子

### 编码器下采样（DSConv）

```
输入: [B, 3, T, 219]
Conv(3→32, k=(1,5), s=(1,2), p=(0,2))
输出: [B, 32, T, 110]

频率轴计算: (219 + 2×2 - 5) / 2 + 1 = 110 ✓
时间轴计算: (T + 0 - 1) / 1 + 1 = T ✓（时间轴不变）
```

### GTConvBlock 的空洞卷积

```
输入: [B, 32, T, 55]
DWConv(32, k=3×3, d=2, p=(2,1))
输出: [B, 32, T, 55]

时间轴感受野: 1 + (3-1)×2 = 5 帧
频率轴感受野: 1 + (3-1)×1 = 3 个频带
```

### 解码器上采样（DSDeconv）

```
输入: [B, 32, T, 55]
ConvTranspose(32→32, k=(1,5), s=(1,2), p=(0,2))
输出: [B, 32, T, 110]

频率轴计算: (55-1)×2 - 2×2 + 5 = 109 → 加 output_padding=1 → 110 ✓
```

## B.6 参数量计算

| 卷积类型 | 参数量公式 |
|---------|-----------|
| 标准卷积 | C_out × C_in × k_h × k_w |
| Depthwise | C × k_h × k_w |
| Pointwise (1×1) | C_out × C_in |
| 深度可分离 | C_in × k² + C_in × C_out |
| GRU（单向） | 3 × (input + hidden) × hidden |
| GRU（双向） | 2 × 3 × (input + hidden) × hidden |
