# FIRFilter_Venture

#### 介绍
FIR滤波器仿真与计算核心，使用纯C++开发，Qt界面做显示

FIRSimulation中的内容为Python对FIR滤波器的仿真，使用Python的scipy库，其中包括各种类型的滤波器，包括Peaking、Low-Pass、High-Pass、Band-Pass、High Shelf,Low ShelfNotch等
